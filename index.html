Meet GraphQL Elixir

GraphQL Elixir is an Elixir library for building and serving GraphQL APIs quickly and easily.

What is GraphQL? GraphQL describes a way of building fluid, single endpoint APIs quickly. We believe GraphQL is going to change how you think about REST APIs.
